# Jignesh Sonewane Portfolio

A personal portfolio website for Jignesh Sonewane, a Cybersecurity Developer and Full-Stack Engineer based in Nagpur, Maharashtra, India.

The site highlights selected work, experience, skills, and contact details in a fast, responsive Next.js app.

## Live Repository

GitHub: <https://github.com/jignesh0462124/Portfolio>

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- GSAP
- Lucide React
- CSS Modules / global CSS

## Features

- Responsive portfolio layout
- Project showcase pages
- Experience and profile sections
- Resume download link
- Contact section
- SEO-friendly app structure with sitemap and robots routes

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open <http://localhost:3000> in your browser.

## Available Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
npm run typecheck
```

## Project Structure

```text
app/                 Next.js app routes and pages
components/          Shared UI and layout components
data/                Portfolio content and site configuration
lib/                 Shared utility components
public/              Static assets, including resume PDF
types/               TypeScript types
```
